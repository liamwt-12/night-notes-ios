# App Store Connect Setup

## In-App Purchases to create:
1. Go to App Store Connect → Night Notes → In-App Purchases
2. Create subscription group called "Night Notes Pro"
3. Add product:
   - Reference name: Night Notes Monthly
   - Product ID: uk.nightnotes.subscription.monthly
   - Price: £4.99/month (Tier 5)
   - Localisation: "Night Notes Pro" / "Unlimited dream interpretations"
4. Add product:
   - Reference name: Night Notes Annual
   - Product ID: uk.nightnotes.subscription.yearly
   - Price: £29.99/year (find nearest tier to £29.99)
   - Localisation: "Night Notes Pro Annual" / "Unlimited dream interpretations"
5. Set uk.nightnotes.subscription.yearly as the upgrade from monthly

## App Store listing:
- Category: Health & Fitness (primary), Lifestyle (secondary)
- Age rating: 4+
- Price: Free (monetised via IAP)
